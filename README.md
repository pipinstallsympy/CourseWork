Данный проект является курсовой работой на 3 курс бакалавриата и основой дипломной работы для 4 курса.

Проект реализует генерацию объекта, проверку на связность материала в нем, восстановление связности и также проверку на перколяцию.

На вход от пользователя ожидается количество разбиений фигуры и количество пор в нем. После генерации и анализа фигуру можно сохранить в удобном .obj+.mtl форматах для дальнейшего анализа/преобразования в других программах.



# Технологический стек:
